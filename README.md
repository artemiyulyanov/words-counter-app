# Words Frequency Report API

Сервис для анализа текстовых файлов и построения статистики по
словоформам.

## Возможности

-   Обработка больших файлов (до GB)
-   Потоковое чтение (без загрузки в память)
-   Нормализация словоформ
-   Генерация Excel (.xlsx)
-   Асинхронная обработка (Celery + Redis)
-   Хранение файлов в S3 (MinIO)

------------------------------------------------------------------------

## Запуск

``` bash
docker-compose up --build
```

------------------------------------------------------------------------

## Доступ

-   API: http://localhost:8000/docs\
-   MinIO: http://localhost:9001\
    login: minioadmin\
    password: minioadmin

------------------------------------------------------------------------

## Тестирование

### 1. Загрузка файла

POST /public/report/export

Body → form-data: - file (.txt)

Ответ:

``` json
{
  "task_id": "..."
}
```

------------------------------------------------------------------------

### 2. Проверка статуса

GET /tasks/{task_id}

Ответ:

``` json
{ "status": "PENDING" }
{ "status": "STARTED" }
{ "status": "done", "download_url": "..." }
{ "status": "error", "error": "..." }
```

------------------------------------------------------------------------

### 3. Скачать результат

Открыть download_url в браузере или сделать GET-запрос.

------------------------------------------------------------------------

## Пример входного файла

    житель живет в городе
    жителем он был долго
    город большой

------------------------------------------------------------------------

## Результат

  Словоформа   Количество   По строкам
  ------------ ------------ ------------
  житель       2            1,1
  город        3            1,0,2
